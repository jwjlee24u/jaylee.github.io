Crpytocurrency Auto-Trading

Raw code on github

Motivation

With increasing interest in investment of Crpytocurrency, I decided to build a auto-trading model to gain profit.

Solution

Using Upbit (Korean Crpytocurrency trading platform) API and AWS cloud server, the trade was automated when it satisfies both of the followings
1. True on Volatility Breakout Strategy by Larry Wiilams where the best K for the strategy was iterated and found as well as the backtest was done before the implementation of the model
2. ML prediction using Prophet was showing Surplus

Volatility Breakout Strategy Diagram (image source)

Results

As the market was a down market, there was no auto trading happened. So tried my best to fix the euqation to get the profit from the down market, but did not work out well.
Eventually, this endeavour led to a loss by 5%.
